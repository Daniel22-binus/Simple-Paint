# Simple-Paint

## **_A simple MVC desktop app for drawing shapes_**

### Features: 
- Drawing,editing and deleting shapes 
- Saving drawings to local drive 
- Undo/redo actions 
- To front/to back actions 
- Log

### Technologies/design patterns implemented in the application: 
- MVC 
- Adapter
- Command
- Serialization
- Prototype
- Observer
